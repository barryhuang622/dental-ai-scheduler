# Dental AI Scheduler

A browser-based prototype for dental clinic scheduling. It supports AI-style one-click scheduling with fairness scoring, human overrides, staff preferences, and CSV export.

## Run Locally

```bash
npm start
```

Then open `http://localhost:3000`.

## Deploy On Railway

Create a Railway project from this GitHub repository. Railway will detect the Node app and run:

```bash
npm start
```
